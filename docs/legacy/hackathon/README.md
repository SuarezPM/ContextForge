# Legacy hackathon artifacts (pre-Apohara rename)

These files come from the V6.0 / V5.0 era when the project was named
`ForgeContextCC` (later `ContextForge`, then renamed to
`Apohara_Context_Forge` in May 2026).

## Files

- `slides.md` — 5-slide marp-compatible pitch deck used in the
  AMD Developer Hackathon 2026 submission.
- `video_script.md` — 451-word, 5-beat 3-minute hackathon demo video
  script.
- `M001-SUMMARY.md` — milestone M001 close artifact (S07/9 era,
  185 hermetic tests passing).
- `M001-LEARNINGS.md` — milestones learnings doc.

## Why kept

These predate the V7 substrate truth-up + MI300X validation work but
are historically valuable: they document the original hackathon
submission narrative. The technical content (test counts, mechanism
descriptions) is superseded by V7.0.0-rc.1 paper v2.0 + AUDIT.md.

Origin repo: `https://github.com/SuarezPM/ForgeContextCC` (different
GitHub repo, archived during the May 2026 rename).
